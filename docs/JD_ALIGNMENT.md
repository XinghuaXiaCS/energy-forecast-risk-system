# JD Alignment

This repository is intentionally mapped to a senior data scientist role that emphasizes:

## Advanced analytics and machine learning
Implemented through binary classification, severity regression, multiclass risk typing, calibration-oriented evaluation, and impact-weighted decisioning.

## Repeatable, interpretable, scalable models
Implemented through reproducible CLI workflows, explicit decision policy logic, reason codes, baseline models, and fallback rules.

## End-to-end lifecycle ownership
Included via ingestion, feature mart, training, scoring, monitoring, API serving, reporting, feedback capture, and retraining triggers.

## Deployment support and ongoing monitoring
Included via Docker, FastAPI, batch CLI, audit logs, data/model/business monitoring, and lifecycle flags.

## Embedding data science into workflows
Included via alert queue, owner assignment, workflow steps, runbook, and cross-functional operating model.
